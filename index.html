CHROME / EDGE EXTENSION

🚀 如何安装扩展

网页版以 Chrome / Edge 扩展形式运行，两个浏览器步骤完全一致，按下面四步即可装好。

⬇ 下载插件
1
打开扩展管理页
地址栏输入 chrome://extensions/（Edge 用 edge://extensions/），或：右上角菜单 → 扩展 → 管理扩展。
2
开启「开发者模式」
页面右上角有「开发者模式」开关，打开它。
3
加载扩展
点击「加载已解压的扩展程序」，选择项目中的 extension/ 文件夹。
4
安装完成！
你会在扩展栏看到 Mineradio 的图标，点击即可开启视觉电台。
